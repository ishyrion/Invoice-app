# InvoiceApp

## Introduction

This is an Angular application. It can be used to invoice products. The user can add new customers, products, quantity and price.

Simple study project.

## How to start?

```
npm
npm install

ng serve
```

## What I used? 

- Angular
- Typescript
- RxJS
- HTML
- SCSS
- Bootstrap
- Jasmine
- Karma
